# Raw Archive

Processed intake. Files land here once their content has been **promoted** into an
authoritative surface (`docs/specs/` ADR or spec, `ROADMAP.md`, a saved memory, or a
`blog/` entry) or has been **superseded**.

Like `_inbox/`, nothing here carries authority — it is a preserved research trail. The
difference is lifecycle: `_inbox/` holds *open* intake still awaiting promotion; this holds
intake whose job is done. Moving a file here clears the active inbox so `/start`'s inbox
check stays meaningful (it flags only genuinely-open intake).

Nothing is deleted on promotion — it is moved here so the reasoning trail survives.
