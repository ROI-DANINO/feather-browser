# Feather Browser

Feather Browser is an exploratory project for a very light, configurable, Chromium-compatible browser with a calm Zen-inspired workflow and a first-class automation layer for agentic AI.

This project follows a phase-gated workflow:

1. Set up the workspace and session tracking.
2. Research the browser architecture options.
3. Plan the first build phase.
4. Build, test, review, and only then plan the next phase.

Only the current phase gets detailed tasks. Later phases stay as roadmap milestones until the previous phase is finished.
