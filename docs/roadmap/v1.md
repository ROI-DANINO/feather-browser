# Feather v1 — "It runs errands for me"

> Front door: [`feather.md`](../../feather.md). Sibling roadmaps: [v2](v2.md), [v3](v3.md).
> Execution index (phases/sessions): [`ROADMAP.md`](../../ROADMAP.md).

**Goal:** tell Feather a task → an agent navigates, figures out the steps, and does it — on normal
sites, plus a first careful try on a scary site (Instagram) with a **throwaway** profile.
Basic "Claude-for-Chrome": real navigation, reasoning about how to do what you ask, running errands.

**Why this is close:** the engine and a warm profile already exist, and an agent (Claude Code) can
drive Feather's local HTTP API today. v1 is mostly *proving and polishing what's already there*, not
building from scratch.

## The first test — the whole point of v1

1. **Create a throwaway Instagram** on the scratch profile (warm via a throwaway Google login; no
   phone number needed). **Collaborative + human-in-the-loop:** the agent fills/navigates, *you* solve
   any CAPTCHA and do the Gmail verification, the agent resumes. *(This is also a manual dry-run of the
   v2 MFA Handler — Feather's window is visible, so "pause for the human" just means you grab the mouse.)*
2. **Smoke test:** "open Instagram, scroll the feed, describe the first 3 posts you see." Pure
   navigate + perceive, nothing that could get flagged.
3. **Level up:** a Social Research errand — open a public profile → read the *visible* comments →
   summarize the vibe. (Own-accounts-only, read what's already visible — not a scraper/bot.)

**Pass** = Feather operates Instagram un-flagged with a real warm session. **Flag** = useful too: it
means v2's stealth hardening is the real next job. (Fallback test then lives in v2: create our own
LinkedIn account.)

> **Expectation:** signup is Instagram's single highest-scrutiny moment — harder than normal browsing.
> A flag *at signup* is not "Feather failed"; it's "the front door is the hardest wall."

---

## Why we can test agent-driving *before* the v2 safety gate

We deliberately test with a **throwaway identity** (scratch profile, throwaway Google) we don't mind
losing. That lets us exercise agent-driving before the real-credential safety gate (v2) is built.
**Real warm profiles stay hands-off until v2.**

## Stealth in v1 = the free layer only

Real Chromium + real warm profile (already the main defense) + human typing cadence + a read-only
self-diagnostic. The heavy hardening and the *gated dangerous surfaces* are v2 — see [v2.md](v2.md).
